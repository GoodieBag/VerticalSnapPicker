# VerticalSnapPicker 
v0.1-beta release.
*Step 1.* Add the JitPack repository to your build file
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

*Step 2.* Add the dependency
```
dependencies {
	        compile 'com.github.GoodieBag:VerticalSnapPicker:v0.1-beta'
	}

```
